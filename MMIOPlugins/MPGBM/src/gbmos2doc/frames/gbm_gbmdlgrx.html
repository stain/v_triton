GBMDLGRX - A REXX adapter for the GBM File Dialog

Introduction

GBMDLGRX is a REXX adapter DLL for OS/2 and eComStation which REXX programs may use to select bitmap filenames to load or save. It uses GBMDLG (GBM File Dialog). Full online help is available via GBMDLG.HLP.

For being able to use it, the following additional files are required:

  • GBM.DLL (GBM Bitmap Format Handler)
  • GBMDLG.DLL (GBM File Dialog)
  • GBMDLG.HLP (GBM File Dialog Help)

Installing GBMDLGRX

Just unzip the GBMOS2PM package and copy the gbmdlg.dll and gbmdlg.hlp files into the same directory as gbmdlgrx.dll. Get the GBMOS2 package and copy gbm.dll into the same directory as gbmdlgrx.dll.

As gbmdlg.dll needs gbm.dll, also gbmdlgrx.dll needs it. gbmdlg.dll and gbm.dll need to be locateable via LIBPATH in order to be able to use gbmdlgrx.dll. If the online help of the GBM File Dialog should be available, gbmdlg.hlp must be locateable via HELP variable.
