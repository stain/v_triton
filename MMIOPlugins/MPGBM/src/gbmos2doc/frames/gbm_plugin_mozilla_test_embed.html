The following table shows a TIF and a PNM image via the GBM plugin for Mozilla. Beside each of the 2 images a reference image is shown by the browser rather than the plugin so that you can easily compare the results.

You need at least GBM.DLL version 1.42. Only then the PNM images will be shown.


The table should be correctly shown by all browsers (also old Netscape)

The TIF and PNM images shown by the GBM plugin are referenced by the EMBED tag which was never part of the HTML standard but is the only way to embed multimedia objects in older browsers. It is still understood by current browsers.


The table should be correctly shown by all newer browsers that understand the OBJECT tag
