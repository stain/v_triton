GBMDLG - File Dialog for OS/2 and eComStation

Introduction

GBMDLG is a DLL which programs may use to select bitmap filenames to load or save. Includes IPF help applicable to the process, and which documents the formats and levels of support given by GBM.

GBMDLG is used by GBMV2 but can be easily used by developers of other programs.

Enhancements in latest GBMDLG compared to the original version

  • English and a German version of the GBM file dialog (applies also to the online context help).
  • Bitmap preview within the file dialog
  • Look & feel has been changed to fit to Warp 4 and eComStation GUI
    • Dialogs now use 9.WarpSans font
    • Resource files for compiling 8.Helv font dialogs are part of the package (use rc.exe)
    • REXX adapter DLL available (GBMDLGRX)

Installing GBMDLG

Just unzip the GBMOS2PM package and copy the gbmdlg.dll and gbmdlg.hlp files into the directory required by the application that can make use of it. Examples of such applications are GBMV2 and Mesa2.

If you've already got the GBMOS2 package installed into a directory, you might as well put these new files in there too.

GBMV2 needs gbm.dll which is part of the GBMOS2 package. Either copy it into the same directory as the application (e.g. gbmv2.exe) or to a directory defined in LIBPATH in the CONFIG.SYS.
